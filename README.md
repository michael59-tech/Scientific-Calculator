# Scientific Calculator App

A Python Tkinter calculator capable of performing:

## Basic Operations
- Addition
- Subtraction
- Multiplication
- Division

## Scientific Features
- sin, cos, tan
- sinh, cosh, tanh
- logarithms
- factorial
- permutations and combinations
- statistical calculations

## Technologies Used
- Python
- Tkinter

## Run the App

```bash
python calculator.py
